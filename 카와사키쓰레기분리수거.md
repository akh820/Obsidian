# 川崎区 日進町 쓰레기 분리수거 완전 정리

## 📋 쓰레기 분리수거 완전 정리표

### 1️⃣ 보통쓰레기 (月·木, 주2회)

| 분류              | ✅ 들어가는 것                                                                       | ❌ 안 들어가는 것                  |
| ----------------- | ------------------------------------------------------------------------------------ | ---------------------------------- |
| **음식물**        | • 음식 찌꺼기 (물기 꼭 제거!)<br>• 조리 후 기름종이<br>• 과일껍질                    | • 캔, 병, 페트병 → 토요일          |
| **종이**          | • 더러운 종이 (음식물 묻은 것)<br>• 화장지, 휴지<br>• 냄새나는 종이                  | • 깨끗한 종이 → 수요일             |
| **비닐/플라스틱** | • 더러운 비닐봉투<br>• プラ마크 없는 비닐<br>• 기름때 안 지워지는 플라스틱           | • プラ마크 있고 깨끗한 것 → 금요일 |
| **옷/천**         | • 옷가지, 천<br>• 인형 (50cm 미만)                                                   | • 50cm 이상 → 대형쓰레기           |
| **깨진 것**       | • 깨진 그릇/접시<br>• 화장품 병<br>• 전구 _(두꺼운 종이에 싸서 "ガラスキケン" 표시)_ | • 음식 담은 유리병 → 토요일        |

### 2️⃣ 캔·페트병·빈병·건전지 (土, 주1회)

| 항목       | ✅ OK                                 | ❌ NG                                                                     | 🤔 헷갈리는 것                                  |
| ---------- | ------------------------------------- | ------------------------------------------------------------------------- | ----------------------------------------------- |
| **페트병** | • 음료수 페트병<br>• 물 페트병        | • 소스/기름 페트병 → 보통쓰레기<br>• 조미료 페트병 → 보통쓰레기           | **캡·라벨 반드시 제거!**<br>헹구고 찌그러뜨려서 |
| **캔**     | • 음료수 캔<br>• 과자 캔<br>• 분유 캔 | • 기름때 안 지워지는 캔 → 보통쓰레기<br>• 스프레이 캔 (가스 완전히 빼고!) | 헹궈서 내놓기                                   |
| **빈병**   | • 음식 담았던 유리병                  | • 화장품 병 → 보통쓰레기                                                  | **뚜껑 제거!** 헹궈서                           |
| **건전지** | • 일반 건전지 (AA, AAA 등)            | • 버튼 전지 → 판매점 회수함<br>• 충전식 배터리 → 판매점 회수함            | 투명 봉투에 따로                                |
