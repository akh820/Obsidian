
| **문장**       | **어미**   | **뉘앙스 및 느낌 (남성 기준)**                              |
| ------------ | -------- | ------------------------------------------------- |
| **俺も行く。**    | (없음)     | **가장 건조함.** 단순한 사실 전달. 감정이 배제된 느낌.                |
| **俺も行くよ。**   | **~よ**   | **정보 전달.** "나도 갈 거야(알고 있어)"라고 상대에게 알려주는 느낌.       |
| **俺도行くね。**   | **~ね**   | **공감/확인.** "나도 갈게, (괜찮지?)"라고 상대의 동의를 구하는 부드러운 느낌. |
| **俺も行くぞ！**   | **~ぞ**   | **강한 선언.** "나도 간다!"라고 스스로 다짐하거나 기합을 넣는 느낌.        |
| **俺も行くぜ。**   | **~ぜ**   | **자신감/멋부림.** "나도 간다구." 친구 사이에서 조금 폼 잡고 싶을 때.      |
| **俺も行くさ。**   | **~さ**   | **당연함.** "나도 당연히 가야지." 혹은 약간 체념하듯 "나도 갈 거야."      |
| **俺も行くかな。**  | **~かな**  | **추측/고민.** "나도 갈까나?" 하고 자신의 행동을 살짝 망설이듯 묻는 느낌.    |
| **俺も行くの？**   | **~의**   | **의문(상승조).** "나도 가는 거야?"라고 부드럽게 물어볼 때.            |
| **俺も行くんだ。**  | **~んだ**  | **설명/단정.** "나도 가는 거야(그러니 토 달지 마)."라는 배경 설명의 느낌.   |
| **俺も行くんだよ。** | **~んだよ** | **강한 설명.** "나도 간단 말이야."라고 자신의 상황을 강조해서 전달할 때.     |
