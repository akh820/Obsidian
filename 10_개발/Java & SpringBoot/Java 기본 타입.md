#Java
- **Java 기본타입**
    - **원시타입(Primitive) : 기본크기 : 기본 값**
        
        **! 소문자로 시작**
        
        1. 정수형
        
        - **byte** : 1byte : 0
        - **short** : 2byte : 0
        - **int** : 4byte : 0
        - **long** : 8byte : 0L
        
        2. 실수형
        
        - **float** : 4byte : 0F
        - **double** : 8byte : 0.0
        
        3. 문자형
        
        - **char** : 2byte : \u0000(유니코드 0번 문자)
        
        4. 논리형
        
        - **boolean** : 1bit : false
        
    - 참조 / 래퍼타입(Reference / Wrapper Type)
        
        원시타입 이외의 모든 타입
        
        **기본값 : null**
