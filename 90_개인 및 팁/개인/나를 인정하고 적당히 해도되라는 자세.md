## **1. 핵심 마인드셋 (가장 중요)**

### **① "있는 그대로도 괜찮다"**

```
지금의 나:
- 완벽하지 않음
- 강박 있음
- 행복하지 않음
- 남 깔보는 경향
- 완성 못 하는 것들

→ 이 모든 것을 인정하기
→ "고쳐야 할 문제"가 아니라 "지금 내 상태"
```

**매일 리마인드:** "나는 불완전하다. 그래도 괜찮다."

---

### **② "70%면 충분하다"**

```
100% = 불가능, 불행
90% = 지속 불가능
70% = 지속 가능, 충분히 좋음
50% = 평균, 괜찮음
30% = 나쁜 날, 그럴 수도
0% = 가끔, 괜찮음
```

**기준:**

- 완벽보다 지속 가능
- 최고보다 충분히 좋음
- 100%보다 매일 50%

---

### **③ "비교는 독이다"**

```
남과 비교 → 우월감/열등감 → 불행
과거의 나와 비교 → "더 잘했어야" → 불행
미래의 나와 비교 → "아직 부족해" → 불행
```

**대신:**

- 어제의 나보다 조금만 나으면 됨
- 남은 남, 나는 나
- 상위 0.1% 될 필요 없음

---

### **④ "과정도 결과다"**

```
❌ 완성 못 하면 = 실패
✅ 시도했으면 = 성공

예:
- 프로젝트 70% = 성공
- 운동 주 2회 = 성공
- 계획 일부만 = 성공
```

**새 기준:** "시작했다" = 이미 50점 "조금이라도 했다" = 70점 "끝까지 했다" = 100점

---

### **⑤ "정답은 없다"**

```
완벽한 운동 루틴 = 없음
완벽한 공부 방법 = 없음
완벽한 식단 = 없음
완벽한 휴식 = 없음
완벽한 삶 = 없음
```

**대신:**

- 시도하고
- 관찰하고
- 조정하고
- 반복하기

### **일하거나 공부할 때:**

**행동:**

- 완벽한 코드 < 작동하는 코드

### **남과 비교하고 싶을 때:**

```
생각: "저 사람은 나보다 나은데..."

1. "내 일이나 하자"
2. "저 사람은 저 사람, 나는 나"
3. "남 깔볼 필요도, 부러워할 필요도 없음"

행동:
- 링크드인/개발 커뮤니티 닫기
- 남의 후기 안 보기
- 내 작업에 집중
```

---

### **완벽하게 못 했을 때:**


**행동:**

- 한 것 세어보기 (안 한 것 세지 말기)
- "그래도 괜찮았어" 소리 내서 말하기
- 자책 금지

---

## **3. 구체적 행동 원칙**

### **"시작-관찰-조정" 사이클**

```
1. 시작: 일단 해보기 (완벽한 계획 없이)
2. 관찰: 어떤가? 편한가? 힘든가?
3. 조정: 너무 쉬우면 올리고, 힘들면 낮추기
4. 반복

예:
- 운동 주 2회 시작
- 관찰: 너무 쉽네?
- 조정: 주 3회로
- 반복
```

---

### **"작게 시작" 원칙**

```
❌ 처음부터 크게
✅ 최소한으로 시작

예:
- 프로젝트: 큰 기능 < 작은 기능 1개
- 운동: 1시간 < 20분
- 공부: 5시간 < 1시간
- 습관: 10개 < 1개
```

---

### **"다시 시작 가능" 원칙**

```
망했어도:
- 내일부터 다시
- 지금부터 다시
- 언제든 다시

예:
- 3일 쉬었어? → 오늘부터
- 한 달 안 했어? → 지금부터
- 완전히 망쳤어? → 내일부터
```

## **마지막 말**

이 모든 것:

- 완벽하게 안 해도 됩니다
- 일부만 해도 됩니다
- 잊어버려도 됩니다
- 다시 강박 와도 됩니다

**이것도 70% 원칙 적용:**

- 이 가이드 70%만 기억해도 충분
- 10%만 실천해도 변화 옵니다
- 완벽하게 안 해도 괜찮습니다

**당신이 할 일:**

1. 이 중에서 **하나**만 골라서
2. 내일부터 해보고
3. 한 달 후 다시 보기

**그게 다입니다.**
